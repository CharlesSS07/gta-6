# Builds

Latest playable builds for the whole team to test.

## Conventions

- Each build folder: `YYYY-MM-DD-<sprint>-<description>/`
- Always include: `build-notes.md` (what's in it, what's broken, how to test)
- Link to QA results from `../qa/` alongside each build
- **Do not commit build artifacts** — link to CDN or deployment URL instead
